# Screenshots

This folder contains the AWS Console screenshots used to document the EBS hands-on lab.

| File | Description |
|---|---|
| `task-02-create-ebs-volume-form.png` | Create an EBS volume and choose size, IOPS, throughput, Availability Zone, and encryption settings. |
| `task-03-volume-actions-attach.png` | Select the available EBS volume and choose **Actions > Attach volume**. |
| `task-03-attach-volume-device.png` | Attach the selected EBS volume to the EC2 instance and choose a Linux device name such as `/dev/sdb`. |
| `task-03-volume-attached-success.png` | Confirm the EBS volume is successfully attached and now shows as `In-use`. |
| `task-06-copy-snapshot.png` | Copy an EBS snapshot within or across AWS Regions. |
| `task-07-create-ami.png` | Create an AMI from the EC2 instance and include attached EBS volume snapshot settings. |
| `task-08-modify-ebs-volume.png` | Modify the EBS volume size from the AWS Console. |
| `task-09-detach-ebs-volume.png` | Detach an EBS volume from the EC2 instance during cleanup. |

Use these screenshots in the main README or task files to show proof of each completed step.
